# shared/moodboard/

**Shared Pixel Journey Visual Reference Library for Marketing Clips**

Static assets (PNG/WEBP) that represent core Px brand elements, characters, UI motifs, and effects. Use these as the foundation for every concept to maintain visual consistency.

**Goal**: One canonical source so every video — whether PxWallet hero, Hot or Not chaos, or Pixal flex — feels like it lives in the same universe.

## Folder Structure

```
shared/moodboard/
├── README.md (this file)
├── SPRINTS-TODO.md (prioritized generation list)
└── assets/
    ├── characters/          # Px Explorer, Whale, Kraken, community meme avatars
    ├── ui-elements/         # Stylized PxWallet screens, Hot or Not voting arena frames, glassmorphic panels
    ├── effects/             # CRT scanlines, subtle bloom, pixel particle presets, glassmorphic overlays
    ├── pixals/              # High-quality renders of example Pixals in different tiers/poses
    ├── color-typography/    # Palette swatches, Press Start 2P samples, text hierarchy examples
    └── backgrounds/         # Retro pixel space scenes, glassmorphic depth layers, arena environments
```

## How to Contribute / Use

1. Add new static references here (not per-concept moodboards — those are for specific video gags or additional lore).
2. Name files descriptively: `px-explorer-idle-pose-01.png`, `hot-or-not-voting-stamp-glass.png`
3. Update `SPRINTS-TODO.md` when you add or complete items.
4. When building a new concept prompt, explicitly list which assets from here to feature.

**All video work starts from these references.** This protects long-term brand coherence while allowing creative concept-specific flourishes in the per-concept moodboard/ folders.

*Part of the Px Perfection Standard for marketing assets.*
